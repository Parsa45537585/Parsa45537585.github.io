فرم ها

  • از فرم ها برای جمع آوری اطلاعات از کاربر و ... استفاده می کنیم.

HOW FORMS WORK

I - A user fills in a form and then presses a button to submit the information to the server.
II - The name of each form control is sent to the server along with the value the user enters or selects.
III - The server processes the information using a programming language uch as PHP, C#, VB.net or Java. It may also store the information in a database
IV - The server creates a new page to send back to the browser based on the information received.

<form> و </form>

  • برای ایجاد یک فرم از تگ form استفاده می کنیم که درون این تگ هم تگ های مختص به خودش برای شکل گیری فرم قرار می گیرد در اصل فرم کانتینری برای المان های دیگر است. 
<form>
    .
    form elements
    .
</form>

Action Atrribute

  • این اتریبیوت که یک اتریبیوت اساسی و اجباری برای المان form است عملی که باید پس از ارائه اطلاعات توسط کاربر انجام شود را تشریح می کند در اصل برای ارسال اطلاعات فرم به سرور استفاده می شود.
  • مقدار این اتریبیوت باید آدرس فایلی باشد که در سرور اطلاعات فرم را دریافت می کند (form-handler). 
<form action="action_page.php">

<input>

  • INPUT یکی از پرکاربرد ترین المان ها در ساخت فرم است که درون المان FORM قرار می گیرد. از تگ input باری ورود اطلاعات توسط کاربر استفاده می شود این اطلاعات می تواند انواع مختلفی چون متن، تاریخ و ... داشته باشد برای تعیین نوع اطلاعات ورودی از اتریبیوت type استفاده می کنیم. در اصل نحوه نمایش المان input بسیار متغیر است و بستگی به اتریبیوت type آن دارد.
  • المان input از void element ها است.
Type Description
<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (for selecting one of many choices)
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button

Attribute های بیشتر برای input

  • Type : با این اتریبیوت پیش تر آشنا شدید.
  • name : این اتریبیوت از اتریبیوت های اجباری است با استفاده از آن یک نام برای المان input تعیین می شود سپس سرور با همین نام اطلاعات وروردی را دریافت می کند (این اتریبیوت برای دیگر المان ها نیز کاربرد دارد).
  • اتریبیوت name برای هر input باید مختلف باشد و تکراری نباشد. 
<input type="text" name="hi">
  • Value : می توان با استفاده از ان یک مقدار پیش فرض برای المان input قرار داد (این اتریبیوت برای تگ های option نیز کاربرد دارد). 
<input type="text" name="hi" value="123">
  • Placeholder : متن نوشته شده درون باکس input. 
<input type="text" name="hi" placeholder="کد ملی">
  • Autocomplete : این اتریبیوت تعیین می کند که به طور اتوماتیک باکس پر شود یا خیر، در اصل به مرورگر اجازه می دهد تا مقدار ورودی را پیش بینی کند و انرا زود تر از آنکه کاربر تکمیلش کند به کاربر پیشنهاد دهد. مقادیر آن نیز yes و no است 
<input type="text" name="code" placeholder="نام" autocomplete="yes">
  • Maxlength : حداکثر کاراکتری که کاربر می تواند وارد المان input کند. 
<input type="text" name="code" placeholder="نام" maxlength="7">
  • Required : برای اجباری کردن پر کردن input از این اترییوت استاده می کنیم این اتریبیوت مقدار ندارد. 
<form>
    <lable for="username">Username:</lable>
    <input type="text" id="username" required>
</form>

<label></label>

  • از این المان برای راهنمایی کاربر که کدام داده را وارد کدام قسمت کند استفاده می شود.
  • این المان یک اتریبیوت به نام for دارد که مقدار آن باید با مقدار id المان مربوطه باشد تا مرورگر بفهمد کدام label مربوط به کدام المان فرم ما است (فقط برای input نیست) 
<form action="/Parsa.php>
    <label for="fname">First name:</label><br>
    <input type="text id="fname" name="fname"><br>
    <label for="lname">Last name:</label><br>
    <input type="text id="lname" name="lname">
</form>



<select> & <option>

  • استفاده از المان select باعث ایجاد لیست انتخابی (drop-down list) می شود که برای قرار دادن هر آیتم در این لیست از المان <option/> و <option> استفاده می کنیم. 
    • <form action="/Parsa.php>
    <label for="a1">مقطع تحصیلی را معین کنید</label>
    <select id="a1" name="paye">
      <option value="دهم">دهم</option>
      <option value="یازدهم">یازدهم</option>
      <option value="دوازدهم">دوازدهم</option>
    </select>
</form>
  • با استفاده از اتریبیوت multiple در تگ select می توانید چند مورد را انتخاب کنید. 
    • <select id="cars" name="cars" size="4" multiple>

<textarea>

  • از این المان برای وارد کردن متن های طولانی مانند نامه ها و انتقادات و پیشنهادات استفاده می کنند.
  • از اتریبیوت rows برای تعیین تعداد خطوط استفاده می شود.
  • از اتریبیوت cols برای تعیین عرض text box ایجاد شده ستفاده می شود.
  • برای تعیین عرض و ارتفاع علاوه بر اتریبیوت از CSS هم می توان استفاده کرد. 
<label style="vertical-align: top;" for="text">انتقادات و پیشنهادات</label>
<textarea name="text1" id="text" cols="20" rows="10">به نظر من</textarea>

<fieldset> & <legend>

  • برای گروه بندی المان های فرم از <fieldset> و برای ایجاد توضیحات برای آن گروه از <legend> استفاده می شود.

  <fieldset>
    <legend>مشخصات تحصیلی</legend>
    <input type="text" placeholder="نام مدرسه">
    <label for="b1">تاریخ شروع تحصیل</label>
    <input type="date" id="b1">
    <label for="a1">مقطع تحصیلی را معین کنید</label>
    <select id="a1" name="paye">
      <option value="دهم">دهم</option>
      <option value="یازدهم">یازدهم</option>
      <option value="دوازدهم">دوازدهم</option>
    </select>
  </fieldset>
</form>
مشخصات تحصیلی

<button>

  • برای قرار دادن دکمه ای که پس از آنکه کاربر فرم را تکمیل کرد با زدن آن از آن اطلاعات را برای سرور ارسال کند از اتریبیوت type استفاده و مقدار آن را submit قرار خواهیم داد. 
    • <button type="submit">ارسال فرم</button>
  • روش دیگر قرار دادن این دکمه این است که از تگ input استفاده کرده و مقدار اتریبیوت type را submit قرار دهیم. 
<input type="submit" value="ارسال فرم">
