عکس در HTML

  • برای اضافه کردن عکس از تگ <img> استفاده می کنیم. این المان Void element است همراه با این المان از اتریبیوت src برای آنکه بگوییم منبع عکس مان کدام فایل باشد استفاده می کنیم. 
<img class="hi" src="hi.jpg" alt="عکسی از طبیعت">
  • آدرس فایل عکس، اگر عکس درون همان فولدری که فایل HTML ما قرار دارد نبود برای آدرس دهی از / استفاده می کنیم.
  • alt اتریبیوتی است که آن را می نویسند تا اگر به هر دلیلی عکس نمایش داده نشد توضیح کوچکی ا آن عکس در دسترس کاربر قرار گیرد.

CSS های المان img

  • width : عرض عکس را تعیین می کند.
  • height : ارتفاع عکس را تعیین می کند.
  • object-fit : اگر دقت کرده باشید با تعیین فقط یکی از property های width و یا height آن یکی هم تغییر می کند، این عملکرد برای این است که shape عکس حفظ شود اما اگر بخواهیم ارتفاع و عرض مورد نظر خود را اعمال کنیم کافی است هر دو property را تنظیم کنیم و در عین حال برای حفظ shape عکس از property به نام object-fit استفاده می کنیم که value های آن به شرح زیر است :
    • cover : باعث می شود عکس بدون کشیدگی تمام ارتفاع و عرض مورد نظر خود را کاور کند.
    • contain : عکس را کوچک می کند تا در محدوده مورد نظر جا شود. 
.hi {width: 400px;
     height: 400px;
     object-fit: cover}
 
.hi {width: 400px;
     height: 400px;
     object-fit: contain}

<div> </div>

  • یکی از مهم ترین تگ های HTML است. این تگ دارای ویژگی های زیر است :
  1. یک تگ container است و تگ های دیگر را در بر می گیرد (حتی div های دیگر).
  2. مانند یک box عمل می کند.
  3. به ما امکان گروه بندی المان ها با یکدیگر را می دهد.
  4. از تگ های block است.

HTML document tree

body = یا جد ancestor

div = یا والد parent

ul, h1, p, hr = یا فرزند child

em, li = یا اولاد descendent

ها li = یا خواهر برادر sibling

انواع تگ های HTML

  • تگ های HTML را می توان به سه دسته تقسیم کرد :
    1. block : این تگ ها کل خط (کل خط container) خود را اشغال می کنند و اجازه نمی دهند المان دیگری در کنار آنان قرار بگیزند.
    2. inline-block : این تگ ها فقط به مقداری صفخه را اشغال می کنند که مورد نیاز آنهاست، در اصل به اندلزه محتویات خود صفخه را اشغال می کنند.
    3. inline : المان هایی که در خط یک متن ظاهر می شوند.
<h1>مجتمع فنی تهران</h1>
<button>sign in</button>
<button>log in</button>

CSS display property

  • برای آنکه کنار المان متنی (p) یک المان دیگر قرار داد باید چه کرد؟ به عبارت دیگر چطور می توان یک تگ که به طور پیش فرض block است را inline-block کرد و یا بلعکس؟ برای انجام این کار در کد های CSS از پراپرتی display استفاده کرده و value های block, inline و یا inline-block قرار می دهیم. 
<head>
    <style>h1{display: inline-block;}</style>
</head>
<body>
<h1>مجتمع فنی تهران</h1>
<button>sign in</button>
<button>log in</button>
</body>

<head>
    <style>button{display: block;}</style>
</head>
<body>
<h1>مجتمع فنی تهران</h1>
<button>sign in</button>
<button>log in</button>
</body>

CSS visiblity

  • از visiblity ، property برای نمایش دادن و یا ندادن المان استفاده می شود. 
    • h2.a {
    visiblity: visible;
}

h2.b {
    visiblity: hidden;
}
  • توجه داشته باشید با قرار دادن مقدار این property بر روی hidden جای خالی المان کماکان نمایش داده می شود و فقط المانی نمایش داده نمی شود.
  • به صورت پیش فرض مقدار این visible , property است.
  • برای آنکه نه المان نمایش داده شود و نه جای خالی آن نمایش داده شود باید از display , proprety و none , value استفاده کنید.

LINKS

  • با ایجاد لینک در فصل اول آشنا شدیم حالا به موارد تکمیلی مربوط به لینک می پردازیم.
  • استفاده از عکس به عنوان لینک : برای استفاده از عکس به عنوان لینک کافی است تگ <img> را درون تگ </a><a> قرار دهیم 
<a href="https://www.digikala.com/"><img src="digikalalogo.png"></a>
  • لینک کردن به آدرس ایمیل : برای این عمل کافی است در مقدار href قبل از آدرس ایمیل از :mailto استفاده کنیم. 
<a href="mailto:someone@examle.com">Send email</a>
  • تلفن : برای قرار دادن شماره تلفن و این قابلیت که به محض کلیک کاربر روی لینک با آن شماره تماس گرفته شود از لینک شماره تلفن استفاده می کنیم با این تفاوت که در مقادیر href قبل نوشتن شماره از :tel استفاده می کنیم. 
<a href="tel:0212729">تماس با مجموعه</a>

Absolute URLs vs. Relative URLs

Absolute

حالتی که آدرس یک سایت بطور کامل در اتریبیوت href قرار بگیرد.

Relative

حالتی که آدرس قسمت دیگری از همان سایت در اتریبیوت href قرار بگیرد.

<p><a href="https://mftplus.com/">سایت مجتمع فنی</a></p>
 
<p><a href="html_images.asp">HTML Images</a></p>

HTML Favicon

  • شاید تا به حال به آیکونی که در کنار نام تب قرار دارد دقت کرده باشید، به این آیکون favicon می گویند. برای قرار دادن این آیکون کافی است در تگ <head> تگ <link> را قرار داده و مقدار اتریبیوت rel را برابر icon گذاشته و مقدار اتریبیوت href را آدرس فایل آیکون مورد نظر قرار دهیم. 
<head>
    <title>Document</title>
    <link rel="icon" href="image/faveicon.ico">
</head>
  • یکی از بهترین سایت ها برای ساخت favicon و یا استفاده از favicon های آماده سایت www.favicon.cc است.
