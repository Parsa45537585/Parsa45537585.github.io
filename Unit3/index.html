سه مدل لیست در html داریم :
Ordered Lists : هر آیتم در لیست دارای شماره است
تگ ها : <ol> و <li>
Unordered Lists : هر آیتم در لیست دارای bullets است
تگ ها : <ul> و <li>
Definition Lists : هر آیتم در لیست توضیحات مختص به خود را دارد
تگ ها : <dl> و <dt> و <dd>

Ordered Lists

  • برای شروع نوشتن این لیست و اینکه به مرورگر بفهمانیم در حال استفاده از این لیست هستیم از تگ <ol> استفاده می کنیم
  • هر آیتمی که می خواهیم در لیست مان باشد باید بین <li> و <li/> نوشته شود 
<body>
    <ol>
        <li>سیب</li>
        <li>انبه</li>
        <li>پرتقال</li>
        <li>نارنگی</li>
    </ol>
</body>
  1. سیب
  2. انبه
  3. پرتقال
  4. نارنگی

CSS برای Ordered Lists

  • برای تغییر نوع شمارش (استفاده از اعداد رومی و ... به جای عدد) از propery , list-style-type در کد های CSS استفاده می کنیم VALUE های این property به شرح زیر است :
<head>
    <syle>ol{list-style-type: lower-roman;}</style>
</head>
<body>
    <ol>
        <li>سیب</li>
        <li>انبه</li>
        <li>پرتقال</li>
        <li>نارنگی</li>
    </ol>
</body>

Definition Lists

  • برای شروع نوشتن این لیست و اینکه به مرورگر بفهمانیم در حال استفاده از این لیست هستیم از تگ <dl> استفاده می کنیم
  • هر آیتمی که می خواهیم در لیست مان باشد باید بین <dt> و </dt> نوشته شود
  • توضیحات هر آیتم باید بلافاصله بعد از تگ های آیتم درون تگ <dd> و </dd> نوشته شود
<body>
    <dl>
        <dt>Sashimi</dt>
        <dd>Sliced raw fish that is served with condiments.</dd>
        <dt>Scale</dt>
        <dd>A device used to measure weight of ingredients.</dd>
        <dt>Scamorze</dt>
        <dd>An Italian cheese usually made from cow's milk.</dd>
    </dl>
</body>

Nested Lists

  • می توان در تگ های <li> پس از نوشتن نام آیتم تگ های <ol> و <ul> قرار داد که موجب زیر لیست می شود و می توان به کمک آن لیستی جدید درون لیست موجود نمایش داد.
<ul>
  <li>متوسطه دوره اول
    <ol>
        <li>هفتم</li>
        <li>هشتم</li>
        <li>نهم</li>
    </ol>
</li>
<li>متوسطه دوره دوم
    <ol>
        <li>دهم</li>
        <li>یازدهم</li>
        <li>دوازدهم</li>
    </ol>
  </li>
</ul>

    2. هشتم
    3. نهم
  • متوسطه دوم
