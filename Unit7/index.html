جدول ها

<table> و </table>

  • برای ساخت یک جدول از المان <table> و </table> استفاده می کنیم، این المان دارای المان های اختصاصی است که درون آن قرار گرفته و جدول زا می سازند.
  • المان هایی که درون المان <table> و </table> قرار می گیرند به شرح زیر است :
    • <tr> </tr>
    <th> </th>
    <td> </td>
    <caption> </caption>

Cell & Rows

  • برای ایجاد هر row در جدول کافی است از تگ <tr> </tr> استفاده کنیم.
  • برای تعریف هر سلول از تگ <td> </td> استفاده کرده و هر چیزی که می خواهیم درون سلول بنویسیم در آن وارد می کنیم. 
<table>
  <tr>
     <td>علی</td>
     <td>قربانی</td>
     <td>123456789</td>
  </tr>
  <tr>
     <td>پارسا</td>
     <td>پورراستگو</td>
     <td>123456789</td>
  </tr>
</table>
علی قربانی 123456789
پارسا پورراستگو 123456789

Table header

  • برای آنکه سلول عنوان را در جدول به کار گرفته و برای اینکه ربای جدول عنوان بگذاریم کافی است به جای المان <td> </td> از المان <th> </th> استفاده کنیم.

Border

  • برای تنظیمات بردر کافی است در کد های CSS ابتدا نام تگ های table, td, th را به عنوان سلکتور (table در بردر دور کل جدول) ، (th, td برای دور هر سلکتور) استفاده می کنیم سپس از پراپرتی border استفاده کرده و مقادیر رنگ، قطر و استایل را برای آن قرار می دهیم. 
<style>
   table, th, td{border: rgb(0, 0, 0) 2px solid;}
</style>

border-collapse

  • برای تنظیم فاصله دار بودن و یا یک دست بودن بردر های جدول از border-collapse property استفاده کرده که دو مقدار زیر را دارا است :
  • Collapse : سلول ها بدون فاصله از هم قرار می گیرند. 
    • <style>
   table, th, td{border: rgb(0, 0, 0) 2px solid; border-collapse: collapse;}

Colspan & Rowspan

  • برای بزرگ کردن به صورت افقی از اتریبیوت colspan در تگ آغازین سلول مورد نظر استفاده و مقدار آن را به اندازه تعداد سلولی می گذاریم که می خواهیم بزرگ شود به این معنا که یک سلول به اندازه چند سلول نرمال بزرگ شود. 
    • <table>
  <caption>1401/09/09</caption>
     <tr>
     <th colspan="3">لیست</th>
  </tr>
  <tr>
     <td>علی</td>
     <td>قربانی</td>
     <td>123456789</td>
  </tr>
  <tr>
     <td>پارسا</td>
     <td>پورراستگو</td>
     <td>123456789</td>
  </tr>
</table>
    1401/09/09
    لیست
    علی قربانی 123456789
    پارسا پورراستگو 123456789

    همانطور که می بینید چون مقدار colspan برای سلول حاوی کله "لیست" برابر 3 است انداز این سلول به صورت افقی به اندازه 3 سلول معمولی است.

  • برای بزرگ کردن به صورت عمودی از اتریبیوت rowspan در تگ آغازین سلول مورد نظر استفاده و مقدار آن را به اندازه تعداد سلولی می گذاریم که می خواهیم بزرگ شود، به این معنا که یک سلول به اندازه چند سلول نرمال بزرگ شود. 
    • <table>
     <tr>
     <th>Name</th>
     <td>Jill</td>
  </tr>
  <tr>
     <th rowspan="2">Phone</th>
     <td>555-1234</td>
  </tr>
  <tr>
     <td>555-8745</td>
  </tr>
</table>
    Name Jill
    Phone 555-1234
    555-8745

    همانطور که می بینید چون مقدار rowspan برای سلول حاوی کلمه "phone" برابر 2 است اندازه این سلول به صورت عمودی اندازه 2 سلول معمولی است.
